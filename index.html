The Next-Gen Operating System

Անվտանգ, արագ և ցածր մակարդակի վերահսկողությամբ օպերացիոն համակարգ՝ կառուցված Rust, C և C++ լեզուներով։ Ստեղծված է ապագայի hardware-ի համար։

Համակարգի Հիմքերը

CellKernel Միջուկ

Մեր սեփական միջուկը, որը կառավարում է պրոցեսները, հիշողությունը և hardware ընդհատումները (interrupts)։

Էջավորում (Paging)

Հիշողության առաջադեմ կառավարում և անվտանգության մակարդակների խիստ սահմանազատում մոդուլների միջև։

Rust & C/C++

Rust-ի հուսալիությունն ու memory-safety-ն միախառնված C և C++ լեզուների բացարձակ արագության հետ։

Կոդավորման Ուղեցույցներ

Ինչպես պատրաստել համակարգը

AweOS-ը հավաքելու համար ձեզ հարկավոր է ունենալ Rust toolchain և հիմնական կոմպիլյատորները։

# Տեղադրել անհրաժեշտ գործիքները
$ rustup target add x86_64-unknown-none
$ sudo apt install build-essential bison flex libgmp3-dev libmpfr-dev libmpc-dev

Միջուկի Build տարբերակը

Անցեք CellKernel թղթապանակ և գործարկեք հետևյալ հրամանները` ISO ֆայլը ստանալու համար։

# Հավաքել նախագիծը
$ cargo build --release
# Ստեղծել bootable ISO հնարավորություն
$ make iso
